# React Table POC

- Collection of hooks to build powerful tables and data grid experiences
- Do not render any markup or styles
- A table utility and not a table component

## Features

- [x] Group headers and footer
- [x] Sorting
- [x] Filtering: Global and Column
- [x] Column Filter
  - disable filters
  - use default column
  - debounce filter
- [x] Pagination
  - Page data and implement next/previous
  - Jump to a page
  - Configure page size
- [x] Select rows
- [x] Column ordering
- [x] Column Hiding
- [x] Stick Columns
- [ ] Grouping
- [ ] In-line row editing
- [ ] Row drag and drop
- [ ] Column re-sizing
- [ ] React Table + Ant Design
